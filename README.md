# nfc-reader

A small Web NFC demo built with React. It shows how to read and write simple text records to NFC tags using the browser's Web NFC API.

## Features
- Read NFC text records using [`readNFC`](src/nfcHandler.js)
- Write NFC text records using [`writeNFC`](src/nfcHandler.js)
- Minimal React UI implemented in [`App`](src/App.js)
- Entry point: [`index.js`](src/index.js)
